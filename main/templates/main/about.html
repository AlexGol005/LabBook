Что это за сайт?

Этот сайт - лабораторная информационная система (ЛИС) под названием Labjournal.

Что умеет сайт:

● Хранить и заполнять список лабораторного оборудования лаборатории (далее ЛО)
● Хранить и заполнять все данные о метрологическом обслуживании ЛО
● Хранить дополнительные сведения об оборудовании (статус, документацию, комментарии и пр.)
● Выгружать список приборов, которые пора поверять
● Выгружать в формате exel формы документов на оборудование, требуемые по ГОСТ 17025 - график поверки, план поверки, отчет по поверке, карточку на прибор, график техобслуживания, этикетку о поверки для наклейки на прибор, план закупки оборудования, протокол верификации нового оборудования и др. 
● Вносить и хранить данные о микроклимате в помещениях, выгружать готовый журнал микроклимата в формате exel


Как можно расширить сайт индивидуально для лаборатории:

● Добавить индивидуальные формы журналов, удобные для конкретной лаборатории.
Для примера - на сайте представлен журнал измерения кинематической вязкости по ГОСТ 33.
Оператору достаточно внести в журнал название пробы время истечения в минутах, журнал сам подставляет данные в формулы и считает конечный результат.
Это исключает ошибку при ручном счёте. Данные измерений за любой период легко доступны через окно поиска. Не надо листать бумажный журнал чтобы найти нужные измерения. Измерения доступны из любой точки мира (при наличии паролей доступа к данным сайта). 
Все результаты можно выгрузить в 2 вариантах - лист, который можно подписать "живой" подписью и подшить в журнал, и протокол измерений, куда автоматически подставляются сведения о поверке использованных средств измерений из списка приборов. 

● Журнал, где отдельно хранятся резульаты измерений - так для каждого измерения возможно увидеть историю, если оно проводилось несколько раз. Возможно установить сроки годности, по истечение которых результат измерения будет подсвечиваться красным цветом.

Чем полезен сайт JL?

● Все данные легко найти через окошки поиска - больше не надо листать кипы бумажных журналов или просматривать огромные файлы exel. При этом поиск доступен даже с мобильного телефона.

● Часто при сдаче приборов на поверку операторы задают разные уточняющие вопросы. При помощи JL можно найти и предоставить всю подробную информацию о каждой единице оборудования прямо с мобильного телефона

● Данные заносятся на сайт 1 раз, и далее из них можно сформировать все необходимые документы. Пример: вам больше не надо будет вписывать сведения о поверке отдельно в график, план, карточку, этикетку, график ТОиР. Достаточно внести данные 1 раз, во всех документах они появятся автоматически.
Это позволяет избежать ситуации, что в одном месте поменяли, в другом забыли, и данные запутались, а также сохраняет ваше время.

● Можно массово поменять формы оформления однотипных документов, а не менять в каждом отдельном документе (например, карточки на приборы).

● При помощи JL можно выгрузить список приборов, которые пора поверять, что сэкономит ваше время и исключит возможность "пропустить" прибор и не отправить в поверку вовремя.

● Можно хранить старые данные так, чтобы они не засоряли актуальные танные, при этом они не удаляются полностью. Пример: списаные приборы из базы не удаляются, но не мешают управлять эксплуатируемыми приборами.

● JL автоматизирует расчеты, что позволяет избежать ошибки по причине "человеческого фактора"

● Все данные защищены от несанкционированного доступа и редактирования. Это значит, что не будет ситуации, когда неизвестный пользователь поменял информацию. Невозможно исправить запись, не имея на это достаточных прав доступа.

● Невозможно внести запись не авторизовавшись на сайте, поэтому у каждой записи автоматически будет указан автор.

● Нет сложной установки и настройки. Вы получаете готовый сайт с дружественным интерфейсом, подобный удобному интернет-магазину.

● JL работает с любого устройства, подключенного к сети интернет, нет требований к компьютеру или операционной системе.
